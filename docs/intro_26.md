**五大核心结论**：
1. **主机厂自研芯片ROI严峻**：仅年销量50万辆+的车企具备经济可行性
2. **"自研"多数是半定制**：200-300人 vs 独立公司1000-2000人
3. **自研不会成为主流**：未来格局"3家深度自研 + 5家半定制 + N家外购"
4. **黑芝麻份额面临压力**：地平线虹吸 + 主机厂自研双重挤压
5. **端侧AI推理是最大增长曲线**：市场规模10×于智驾芯片
